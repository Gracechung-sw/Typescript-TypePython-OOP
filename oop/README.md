## 1. 객체지향 concept

절차지향 - 함수가 막 얽혀있고, 데이터가 다른 곳에서 막 변경되곤 하기 때문에 하나를 수정하기 위해서는 전체적인 application를 이해하고 있어야 함

객체지향 - 서로 관련있는 데이터.. 어쩌구 객체지향과 절차지향에 대해 정의 알고, 비교하기
